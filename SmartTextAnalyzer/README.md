# Smart Text Analyzer

A simple Python program that analyzes text input and provides various statistics about the text.

## Features

- Counts the number of characters in the text
- Counts the number of words in the text
- Identifies the longest word in the text
- Counts the number of vowels in the text
- Displays the text in both uppercase and lowercase versions
- Shows word frequency distribution

## How to Use

Run the program and enter a sentence when prompted. The program will analyze the text and display all the statistics mentioned above.

## Requirements

- Python 3.x
